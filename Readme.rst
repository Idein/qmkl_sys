
qmkl_sys
================================================================

This library provides a thin Rust interface to `qmkl`_ via the Foreign Function Interface(FFI) .


Requirements
----------------------------------------------------------------

- libclang
- qmkl


Build
----------------------------------------------------------------

.. code-block:: shell

    $ docker-compose run --rm build-qmkl-sys


.. _qmkl: http://github.com/Idein/qmkl
